# Plugins Overview

Welcome to the plugins documentation. Here you'll find information about all available plugins for the library.

## Available Plugins

### Authentication Plugin
Complete authentication solution with OAuth, SAML, and custom providers.

[View Documentation →](auth/index.html)

### Cache Plugin
High-performance caching layer with multiple backend support.

[View Documentation →](cache/index.html)

## Installation

All plugins can be installed via npm:

```bash
npm install @mylib/plugin-auth
npm install @mylib/plugin-cache
```
