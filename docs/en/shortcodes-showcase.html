Shortcodes Showcase

This page demonstrates all available shortcodes in Chiron. Use this as a reference and testing ground.

What are Shortcodes?

Shortcodes are WordPress-compatible syntax elements that let you add rich components to your markdown content. They're processed before markdown parsing, so you can use markdown inside them!

Tabs

Perfect for code examples, installation guides, or platform-specific instructions.

// JavaScript example
const greeting = 'Hello, World!';
console.log(greeting);
# Python example
greeting = 'Hello, World!'
print(greeting)
# Ruby example
greeting = 'Hello, World!'
puts greeting

Accordion

Great for FAQs, detailed explanations, or collapsible content.

What is Chiron?

Chiron is a modern static site generator designed specifically for documentation. It combines the simplicity of markdown with powerful features like shortcodes, i18n support, and advanced theming.

Why use shortcodes?

Shortcodes provide a WordPress-compatible way to add rich components without writing HTML. They're:

  • Easy to use
  • Secure by default
  • Extensible
  • Familiar to millions of users
Can I create custom shortcodes?

Yes! You can register custom shortcodes programmatically. See the documentation for details.

FAQ

What is rate limiting?

Rate limiting restricts the number of API calls you can make per hour. Free tier: 100 requests/hour. Pro tier: 10,000 requests/hour.

How do I upgrade my plan?

Visit your dashboard and click "Upgrade Plan" to see available options.

Is there a sandbox environment?

Yes! Use https://sandbox.api.example.com for testing without affecting production data.
