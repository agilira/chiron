Data Island Pattern Demo

This page demonstrates the Data Island Pattern - injecting data at build-time for lazy-loaded apps.

How it Works

  1. Build Time: Chiron generates a <script type="application/json"> with page data
  2. Load Time: The app script loads only when scrolled into view (Intersection Observer)
  3. Runtime: App reads data from JSON script - zero HTTP requests!

Technical Details

The data attribute in the shortcode specifies which variables to inject:

<App data="page.title,config.site.name" />

Chiron generates:

<script type="application/json" id="my-app-data">
{
  "title": "Page Title",
  "name": "Site Name"
}
</script>

The app reads it with:

const data = JSON.parse(
  document.getElementById('my-app-data').textContent
);

Zero runtime overhead. Maximum performance. üéØ
