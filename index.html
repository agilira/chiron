Chiron

A modern, accessible documentation template for microsites. Perfect when you need more than a README but less than a full documentation site.

Overview

Chiron is a README on steroids - the perfect solution when your project needs more than a simple README but less than a full documentation site. Built for microsites, it provides professional, accessible documentation without the complexity of build processes.

Modern Design

Clean, professional interface with responsive design that works seamlessly across all devices and screen sizes.

Accessibility

WCAG-compliant design with full keyboard navigation, screen reader support, and inclusive user experience.

Customizable

Flexible configuration system allows easy customization of branding, navigation, and features to match your project needs.
